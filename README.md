
# Text Editor

A Text Editor Made in C++ with multiple features

    
## Features

- New Window
- Open a file
- Save/Save as
- Change FileType for saving
- Print
- Built in Calcualtor
- Text to Speech built in
- Insert Math symbols
- Theme Selection
- Built in Music Player
- Custom MenuStrip
